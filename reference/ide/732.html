FLOOR

FLOOR (x)

Largest integer value not greater than x.

For largest integer value greater then x see CEIL(x)

print "floor(1.1)  = "; floor(1.1)
print "floor(1.5)  = "; floor(1.5)
print "floor(1.9)  = "; floor(1.9)
print "floor(-1.1) = "; floor(-1.1)
print "floor(-1.5) = "; floor(-1.5)
print "floor(-1.9) = "; floor(-1.9)
