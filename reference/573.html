DATEDMY

DATEDMY dmy| julian_date, BYREF d, BYREF m, BYREF y

Home > Reference > Date

Returns the day, month and the year as integers.

See also DATE.

DateString = date
datedmy DateString, d, m, y
print DateString + " -> Day: " + d + " Month: " + m + " Year: " + y
