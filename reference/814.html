TRON

TRON

Home > Reference > System

Turns on the trace mechanism. When trace mechanism is ON, SmallBASIC displays each line number as the program is executed.

See TROFF to switch off trace mechanism.

Example

tron
for ii = 1 to 5
next

' Output: <2><3><3><3><3><3>
