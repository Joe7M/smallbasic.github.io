DATE

s = DATE

Home > Reference > Date

Returns the current date as string with format “DD/MM/YYYY”.

See DATEDMY for splitting a date-string to integers.

Example

DateString = date
datedmy DateString, d, m, y
print DateString + " -> Day: " + d + " Month: " + m + " Year: " + y
