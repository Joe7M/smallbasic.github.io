NIL

a = NIL

Home > Reference > System

Sets a variable to nil. NIL is used to mean ‘not set’ as distinct from having a number set to 0.

Example 1

b = NIL
print b   ' Output: Nil

Example 2

b = NIL
if(b == NIL) then print "b is not set"

' Output: b is not set
