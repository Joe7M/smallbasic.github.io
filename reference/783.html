LEFT

ls = LEFT (s [,n])

Home > Reference > String

Returns the n number of leftmost chars of string s as string ls. If n is not specified then n = 1.

Example

s = "abcdef"
print LEFT(s)      ' Output: a
print LEFT(s, 3)   ' Output: abc
