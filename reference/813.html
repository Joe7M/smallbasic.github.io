TROFF

TROFF

Home > Reference > System

Turns off the trace mechanism. When trace mechanism is ON, SmallBASIC displays each line number as the program is executed.

See TRON to switch on the trace mechanism.

Example

tron
for ii = 1 to 5
next

troff
for ii = 1 to 5
next
