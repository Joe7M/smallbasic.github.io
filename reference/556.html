ISDIR

b = ISDIR (dir)

Home > Reference > Data

Returns true if the string dir is a directory.

Example

print IsDir("C:\users")        ' Output 1 on Windows, otherwise 0
print IsDir("/home")           ' Output 1 on Linux, otherwise 0
