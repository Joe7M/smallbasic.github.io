Language

LET var=expr

Assigns a value to the variable.


* var - A valid variable name.
* expr - The value assigned to variable.
LET x = 4
x = 1               ' Without the LET keyword
z = "String data"   ' Assign string
...
DIM v(4)
z=v                 ' Assign array (z = clone of v)
