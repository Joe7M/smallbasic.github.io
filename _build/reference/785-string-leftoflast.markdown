# LEFTOFLAST

> s = LEFTOFLAST (s1, s2)

Returns the left part of `s1` at the position of the last occurrence of the string `s2` in string `s1`. If `s2` does not occur in `s1`, an empty string will be returned.

### Example

```
s1 = "abcdefdeg"
s2 = "de"
print LEFTOFLAST(s1, s2)      ' Output: abcdef
```
