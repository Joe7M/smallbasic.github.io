# CEIL

> n = CEIL (x)

Smallest integer value not less than `x`.

For smallest integer value less then `x` see FLOOR.

### Example

```
print "ceil(1.1)  = "; ceil(1.1)
print "ceil(1.5)  = "; ceil(1.5)
print "ceil(1.9)  = "; ceil(1.9)
print "ceil(-1.1) = "; ceil(-1.1)
print "ceil(-1.5) = "; ceil(-1.5)
print "ceil(-1.9) = "; ceil(-1.9)

' Output:
' ceil(1.1)  = 2
' ceil(1.5)  = 2
' ceil(1.9)  = 2
' ceil(-1.1) = -1
' ceil(-1.5) = -1
' ceil(-1.9) = -1
```
