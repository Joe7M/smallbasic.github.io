# EMPTY

> EMPTY (x)

Returns true if x is
* a zero length array
* an empty string
* a number with the value 0.

### Example

```
s = ""
a = []
dim b
i = 0

if(empty(s)) then print "s is empty"
if(empty(a)) then print "a is empty"
if(empty(b)) then print "b is empty"
if(empty(i)) then print "i is empty"
```
