# SUMSQ

> f = SUMSQ (var1 [, ..., varN])

Sum of square value(s) `var1` to `varN`. `var1` to `varN` are single numbers or arrays.

### Example

```
A = [1, 2, 3]
print sumsq(1, 2, 3)      ' 1^2 + 2^2 + 3^2 -> Output: 14
print sumsq(A)            ' Output: 14
print sumsq(A, 1, 2, 3)   ' Output: 28
```
